### Proyecto integrador

Durante el mes de octubre y noviembre, se hicieron relevamiento en diferentes áreas dentro del colegio, a fin de detectar y poder darle alguna solución informática a los problemas que los mismos fueron trasmitiendo.

### Herramientas utilizadas

- [Django](https://www.djangoproject.com/)
- [Django Rest Framework](https://www.django-rest-framework.org/)
- [Bootstrap](https://getbootstrap.com/)
- [Crispy Forms](https://django-crispy-forms.readthedocs.io/en/latest/)
- [Django CKEditor](https://django-ckeditor.readthedocs.io/en/latest/)
- [Pillow](https://pillow.readthedocs.io/en/stable/)
